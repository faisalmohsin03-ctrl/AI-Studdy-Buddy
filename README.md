🤖 AI Study Buddy — Agentic AI Learning Assistant

📌 Project Overview

AI Study Buddy is an Agentic AI system designed to provide personalized learning support automatically through email-based interaction.

The system receives student questions via email, processes them using Large Language Models (LLMs), and sends structured study responses including summaries, keywords, and real-world analogies.

Aligned with UN SDG 4 – Quality Education, the project aims to make learning accessible, adaptive, and intelligent.

---

⚙️ How It Works

1. Student sends question through Email
2. AI Agent detects incoming query
3. LLM performs contextual reasoning
4. Generates simplified notes & explanations
5. Automatically replies via Email
